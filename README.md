# EJERCICIOS Y EVALUACIONES.
Programación orientada a Objetos, Java, 2024.
Universidad del Bío-Bío

## Descripción
Este repositorio contiene ejercicios y evaluaciones de la asignatura Programación orientada a Objetos, 
realizados durante el año 2024.

Este repositorio se mantiene como un archivo historico, dirigido a futuros estudiantes de la asignatura, 
que necesiten ejercicios y evaluaciones para practicar.

Este repositorio NO esta dirigido para copiar o plagiar ejercicios y evaluaciones, sino para practicar. El
correcto uso del repositorio es responsabilidad de cada estudiante.

## Contenido
El contenido se encuentra dentro de la carpeta **"src"**. Dentro de esta carpeta se encuentran los **ejercicios** 
y las **evaluaciones**.

Cabe destacar que los ejercicios y evaluaciones se encuentran en carpetas separadas.

Es importante señalar varios ejercicios y evaluaciones NO son enteramente funcionales, y fueron guardados para
servir como ejemplo de lo que se espera en la asignatura.
